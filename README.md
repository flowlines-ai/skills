# Flowlines Skills for Coding Agents

A collection of skills that help AI coding agents integrate the Flowlines SDK into your projects.

## Available Skills

| Skill | Description |
|---|---|
| `integrate-flowlines-sdk-js` | Integrates the `@flowlines/sdk` into a Node.js/TypeScript codebase |
| `integrate-flowlines-sdk-python` | Integrates the `flowlines` SDK into a Python codebase |
| `integrate-flowlines-opencode-plugin` | Integrates the `@flowlines/opencode-plugin` into an OpenCode project |

## Installation

```bash
npx skills add flowlines-ai/skills
```
